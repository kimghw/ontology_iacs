# Negative depth correction (Regulation 31(3))

**(1972)**
**(Rev.1**
**July 2008)**

When the height of a superstructure, raised quarterdeck or trunk is less than the corresponding standard height, it is recommended that the calculated reduction be corrected in the ratio of the height of the actual superstructure, raised quarterdeck or trunk to the applicable standard height as defined in Regulation 33.

---

**Footnote:** This UI is also applicable to Regulation 31(3) of the 1988 Protocol and the revised 1988 Protocol.
