# LL10 Air pipes (Regulation 20)

(1968)
(Rev.1 July 2008)
(Rev.2 Nov 2020)

For ships assigned timber freeboards the air pipes should be provided with automatic closing appliances.

---

Note:
This UI is applicable to Regulation 20 of the International Convention on Load Lines, 1966 and 1988 Protocol.
