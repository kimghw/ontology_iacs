# LL43 Minimum bow height (Regulation 39)

(1978)
(Rev.1
July 2008)

On ships to which timber freeboards are assigned Regulation 39 should relate to the summer load waterline and not to the timber summer load waterline.

---

**Footnote:** This UI is also applicable to Regulation 39 of the 1988 Protocol.
