# LL56 Block Coefficient of a Pontoon (Regulation 3(7))

(1993)
(Rev.1
July 2008)

When calculating the block coefficient of a pontoon, according to the definition of Regulation 3(7), the 96%-length, if applicable, shall be used disregarding the fact that this might lead to a coefficient greater than 1,0.

---

Footnote: This UI is also applicable to Regulation 3(7) of the 1988 Protocol.
