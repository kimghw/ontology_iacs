# LL1

## Application (Article (4))

(1968)
(Rev.1
July 2008)

Even where the increase in draught is only of the order of 1 in or 2 in there should be no relaxation from the condition that existing ships comply with all the requirements.

---

**Footnote:** This UI is also applicable to the revised Article 4 of the 1966 Convention as modified by its 1988 Protocol.
