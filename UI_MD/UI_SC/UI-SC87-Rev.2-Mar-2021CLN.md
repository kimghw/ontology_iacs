# SC87 Certification of Carriage of Solid Dangerous Bulk Cargoes
(1993)
(Rev.1 Nov 2005)
(Rev.2 Mar 2021)

**SOLAS Regulation II-2/19.3 and 19.4**

Certification for carriage of solid dangerous bulk cargoes covers only those cargoes listed in Group B in the IMSBC Code except cargoes classified solely as MHB. Other solid dangerous bulk cargoes may only be permitted subject to acceptance by the Administrations involved.
(MSC/Circ. 1120)

Note:
1. Rev.2 of the UI is editorially amended to refer to the IMSBC Code.
