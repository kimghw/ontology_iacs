# SC175 Combustible Gaskets in Ventilation Duct Connections (Reg.II-2/9.7.1.1)
(July 2003)

Combustible gaskets in flanged ventilation duct connections are not permitted within 600 mm of an opening in an A class or B class division and in ducts required to be of A class construction.

Note:
1. This UI is to be uniformly implemented by IACS Members and Associates from 1 January 2004.
