# SC 158

## SC158 Horizontal fire zone concept

(June 2000)
(Rev.1 Nov 2005)

**(Reg. II-2/20.2.2.1)**

The "Total overall clear height" is the sum of distances between deck and web frames of the decks forming one horizontal zone.
(MSC/Circ. 1120)

**Note:** This UI SC 158 is to be uniformly implemented by IACS Members and Associates from 1 January 2001.
