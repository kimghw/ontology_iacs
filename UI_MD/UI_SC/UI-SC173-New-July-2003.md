# SC173 Safety Devices in Venting Systems
(July 2003) (Reg.II-2/4.5.3.3)

Ullage openings do not include cargo tank openings that are fitted with standpipe arrangements with its own manually operated shutoff valves.

Examples include the common 1" to 2" diameter standpipe arrangements that are used for sampling, monitoring or measuring of ullage/temperature/interface, oxygen, liquid and hand dipping in the cargo tank.

Note:
1. This UI is to be uniformly implemented by IACS Members and Associates from 1 January 2004.
