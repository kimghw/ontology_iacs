# SC174 A 60 Front Insulation of Tankers (Reg.II-2/9.2.4.2.5)

(July 2003)
(Rev.1 Aug 2006)

For the portions which face the cargo area, the "A-60" class insulation should be provided up to the underside of the deck of the navigation bridge.

(MSC.1/Circ.1203)

Note:
1. This UI is to be uniformly implemented by IACS Members and Associates from 1 January 2004.
2. Rev.1 of this UI is editorially amended to refer to MSC.1/Circ.1203.
