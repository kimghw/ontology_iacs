# SC45 Fire integrity of bulkheads and decks
(Rev.1 Nov 2005)

(Reg. II-2/9.2.3 and 9.2.4)

The following spaces are considered to belong to the categories of spaces dealt with by Reg. II-2/9.2.3 and 9.2.4, for the purpose of this regulation, as follows:

*   Navigation equipment room (radar transmitter) and battery rooms (1): Control Stations

Note 1: Provision chambers are to be treated as store rooms.

Note 2: Refrigerated provision chambers are to be Category 9 service spaces if thermally insulated with combustible materials, or Category 5 service spaces if thermally insulated with non-combustible materials.
