**CC2 Interpretation of paragraph 4.9.2, BCH Code**
(1977)
(Rev.1
Feb
2007)
(corresponds to paragraph 15.12.2 of the IBC Code)

Paragraph 4.9.2 of the Code reads as follows:

'Tank venting systems should be provided with a connection for a vapour return line to shore installation.'

This paragraph may be interpreted as follows in respect of the provision of a stop valve for the connection of tank venting systems with lines for the return of vapours to shore plants:

'Tank venting systems should be provided with a stop valve for vapour return line to shore.'
