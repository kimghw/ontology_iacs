(July 2004)

### Regulation 16 (7)

Deleted Nov 2013.

It should be noted that the layout of reg 16 in total was revised by MEPC.176(58) in order to provide a more logical arrangement.

Hence Reg 16.7 which now covers this point makes specific reference to only being applicable to those incinerators required by Reg 16.6.1.
