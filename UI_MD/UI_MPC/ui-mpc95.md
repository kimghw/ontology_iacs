## SC224 Measurement of Distances
(Aug 2008)

### LL74
(Aug 2008)
Several IMO instruments (e.g., ICLL, SOLAS and MARPOL Conventions, the IBC Code and the IGC Code, etc.) require distances to be measured such as tank length, height, width, ship (or subdivision or waterline) length, etc..

## MPC95
(Aug 2008)
Interpretation

Unless explicitly stipulated otherwise in the text of the regulations in SOLAS, Load Line and MARPOL Conventions and any of their mandatory Codes, distances are to be measured by using moulded dimensions.

Note:
This Unified Interpretation is to be uniformly implemented by IACS Societies from 1 April 2009.
