# Procedure for handling technical questions regarding the IACS CSR

**No. 32**
(Rev.0 July 2009)
(Corr.1 Sept 2009)
(Rev.1 Sept 2011)
(Rev.2 Nov 2024)

1. The obligations of this Procedural Requirement apply to Classification Societies (CS) which have embedded the IACS Common Structural Rules for Bulk Carriers and Oil Tankers into their classification Rules.

2. Each CS is responsible for internal procedures that provide the required framework for consistent implementation of the IACS CSR.

3. Each CS is to have a single point of contact to deal with the IACS CSR Maintenance PTs.

4. When a CS receives questions, comments or requests regarding the IACS CSR, the CS is to first access the IACS CSR Knowledge Centre (KC) to find out whether the same or similar issue and its solution have already been documented.

4.1 If a solution is found in the CSR KC, the CS is to reply to the initiator with the existing solution.

4.2 In case the issue is simple or non-controversial, the CS may reply to the initiator at the CS's own responsibility.

4.3 For issues involving new interpretation, questions sent to more than one Society, consideration of equivalency, or questions requiring a Rule Change, the "Procedures for the CSR Maintenance" are to be followed.

**End of Document**
