## F20.1 일반 요건

F20.1.1 모든 유형의 불활성 가스 장치는 다음을 준수하여야 한다:

.1 다음 사항을 포함하는 계통도 형식의 도면을 심사를 위해 제출하여야 한다:
- 모든 제어 및 감시 장치를 포함한 불활성 가스 생성 장치의 상세 및 배치;
- 불활성 가스 분배를 위한 배관 계통의 배치.

.2 모든 운전 조건에서 적합한 불활성 가스를 생산할 수 있는 자동 제어 장치를 설치하여야 한다.

.3 후속 검사는 선급 규칙에서 요구하는 간격으로 실시하여야 한다.

## F20.2 SOLAS 규칙 II-2/4.5.5.1이 적용되는 탱커(케미컬 탱커 포함)의 모든 장치에 대한 요건

F20.2.1 SOLAS 규칙 II-2/4.5.5.1이 적용되는 탱커에는 MSC.367(93)에 의해 개정된 FSS Code 제15장의 해당 요건에 적합한 불활성 가스 장치를 설치하여야 한다. FSS Code 제15장의 해당 요건 적용 시, 그 안의 "주관청(Administration)"이라는 용어는 해당 선급으로 간주한다. 불활성 가스 장치는 MSC.365(93)에 의해 개정된 SOLAS 규칙 II-2/16.3.3에 따라 운용하여야 한다. SOLAS 규칙 II-2/16.3.3.2 적용 시, FSS Code 제15장 2.2.1.2.4항을 준수하여야 한다.

## F20.3 SOLAS 규칙 II-2/4.5.5.1이 적용되는 탱커(케미컬 탱커 포함)의 질소 발생 장치에 대한 추가 요건

F20.3.1 다음 요건은 SOLAS 규칙 II-2/4.5.5.1에 의해 요구되는 바에 따라 선박에 질소 발생 장치가 설치되는 경우에 적용된다. 이 목적을 위해, 불활성 가스는 압축 공기를 중공사 섬유 다발, 반투과성 막 또는 흡착 물질을 통과시켜 공기를 구성 가스로 분리하여 생산하여야 한다.

F20.3.2 MSC.367(93)에 의해 개정된 FSS Code 제15장의 해당 요건에 추가하여, 질소 발생 장치는 SOLAS 규칙 II-2/4.5.3.4.2, 4.5.6.3 및 11.6.3.4를 준수하여야 한다.

F20.3.3 질소 발생기는 급기 처리 장치와 MSC.367(93)에 의해 개정된 FSS Code 제15장 2.2.1.2.4항을 충족하는 데 필요한 수의 병렬 배치된 막 또는 흡착 모듈로 구성되어야 한다.

F20.3.4 질소 발생기는 MSC.367(93)에 의해 개정된 FSS Code 제15장 2.2.1.2.5항에 따라 고순도 질소를 공급할 수 있어야 한다. MSC.367(93)에 의해 개정된 FSS Code 제15장 2.2.2.4항에 추가하여, 장치에는 시동 및 비정상 운전 중에 "규격 외(off-spec)" 가스를 대기 중으로 배출하는 자동 수단을 설치하여야 한다.

F20.3.5 장치에는 MSC.367(93)에 의해 개정된 FSS Code 2.2.1.2항에서 요구하는 총 가스량을 공급할 수 있는 충분한 양압을 생성하기 위한 하나 이상의 압축기를 설치하여야 한다. 두 대의 압축기가 설치되는 경우, 장치의 총 요구 용량은 두 압축기에 균등하게 배분하는 것이 바람직하며, 어떠한 경우에도 한 대의 압축기 용량이 요구 총 용량의 1/3 미만이어서는 안 된다.

F20.3.6 MSC.367(93)에 의해 개정된 FSS Code 제15장 2.4.1.2항에서 요구하는 바와 같이 압축 공기에서 유리수, 입자 및 미량의 유분을 제거하기 위해 설치된 급기 처리 장치는 규정 온도도 유지하여야 한다.

F20.3.7 질소 발생기로부터의 산소 농축 공기 및 질소 수신기의 보호 장치로부터의 질소 농축 가스는 노출 갑판의 안전한 위치로 배출하여야 한다.

F20.3.8 정비를 위해 발생기와 수신기 사이에 격리 수단을 설치하여야 한다.

---

**안전한 위치의 정의:**

1. 질소 발생기로부터의 산소 농축 공기 - 노출 갑판의 안전한 위치는 다음과 같다:
   - 위험 구역 외부;
   - 인원 통행 구역으로부터 3m 이내가 아닌 곳; 및
   - 기관(엔진 및 보일러)용 공기 흡입구 및 모든 환기 흡입구로부터 6m 이내가 아닌 곳.

2. 질소 수신기의 보호 장치로부터의 질소 농축 가스 - 노출 갑판의 안전한 위치는 다음과 같다:
   - 인원 통행 구역으로부터 3m 이내가 아닌 곳; 및
   - 기관(엔진 및 보일러)용 공기 흡입구 및 모든 환기 흡입구/배출구로부터 6m 이내가 아닌 곳.

## F20.4 SOLAS 규칙 II-2/4.5.5.1 및 4.5.5.2에서 요구하는 불활성화 이외의 목적으로 설치되는 질소/불활성 가스 장치

F20.4.1 본 절은 SOLAS 규칙 II-2/4.5.5.1 및 4.5.5.2가 적용되지 않는 유조선, 가스 운반선 또는 케미컬 탱커에 설치되는 장치에 적용된다.

F20.4.2 MSC.367(93)에 의해 개정된 FSS Code 제15장의 2.2.2.2, 2.2.2.4, 2.2.4.2, 2.2.4.3, 2.2.4.5.1.1, 2.2.4.5.1.2, 2.2.4.5.4, 2.4.1.1, 2.4.1.2, 2.4.1.3, 2.4.1.4, 2.4.2.1 및 2.4.2.2항이 해당되는 경우 장치에 적용된다.

F20.4.3 F20.3절의 요건이 적용되나, F20.3.1, F20.3.2, F20.3.3 및 F20.3.5항은 제외한다.

F20.4.4 불활성 가스 장치에 사용되는 재료는 선급 규칙에 따라 의도된 목적에 적합하여야 한다.

F20.4.5 모든 장비는 선상에 설치되어 검사원이 만족하는 조건에서 작동 시험을 실시하여야 한다.

F20.4.6 MSC.367(93)에 의해 개정된 FSS Code 제15장 2.2.3.1.1항에서 요구하는 두 개의 역류 방지 장치는 불활성 가스 주관에 설치하여야 한다. 역류 방지 장치는 MSC.367(93)에 의해 개정된 FSS Code 제15장 2.2.3.1.2 및 2.2.3.1.3항을 준수하여야 한다. 다만, 화물탱크, 화물창 또는 화물 배관으로의 연결이 영구적이지 않은 경우, MSC.367(93)에 의해 개정된 FSS Code 제15장 2.2.3.1.1항에서 요구하는 역류 방지 장치는 두 개의 역류 방지 밸브로 대체할 수 있다.
