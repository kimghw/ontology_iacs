## G3.1 일반사항

**G3.1.1** 본 규정은 선급 목적의 액화가스 운반선 관련 항목에 대한 승인 및 검사를 위한 일반 원칙을 규정한다.

**G3.1.3** IGC Code란 산적 액화가스 운반선의 건조 및 설비에 관한 국제코드(IMO 결의 MSC.370(93), MSC.411(97) 및 MSC.441(99)에 의해 개정됨)를 말한다.

---

## G3.2 적용범위

본 요건은 화물 가스 배관 및 안전밸브 배출관을 포함한 액화가스 화물 및 공정 배관에 적용된다.

---

## G3.3 내압에 대한 부재치수

### G3.3.2 설계 압력
운반 화물에 따라, IGC Code 4.13.2.2항 및 5.4항에 따른 다음 조건들 중 최대값을 사용해야 한다.

### G3.3.3 허용응력
배관의 경우, 허용응력 K는 IGC Code 5.11.3.1항에 정의된 값 중 작은 값으로 한다.

### G3.3.4 최소 벽두께

$$t = \frac{t_0 + b + c}{1 - \frac{a}{100}}$$

여기서:
- t₀ = PD/(2Ke + P) (이론 두께)
- P = 설계 압력 (MPa)
- D = 외경 (mm)
- K = 허용응력 (N/mm²)
- e = 효율계수 (이음매 없는 관/승인된 용접관의 경우 1)
- b = 굽힘에 대한 여유 = (1/2.5)(D/r)t₀
- c = 부식여유 (mm)
- a = 제조 마이너스 공차 (%)

### G3.3.5 플랜지, 밸브, 관이음쇠
IGC Code 5.4항에 따른 설계 압력을 고려하여 공인 표준에 따른다.

---

## G3.4 응력해석

**G3.4.1** 설계 온도가 ≤ -110°C일 때 중량, 내압, 열수축 및 선체 변형을 고려한 완전 응력해석을 수행한다.

---

## G3.5 재료

**G3.5.1** 최소 설계 온도를 고려하여 IGC Code 5.12.1항, 5.12.2항 및 W1에 따른다.

---

## G3.6 선내 설치 전 배관 구성품 및 펌프의 시험

### G3.6.1 밸브

**G3.6.1.1 원형시험(Prototype Testing):** -55°C 미만의 각 유형에 대해 IGC Code 5.13.1.1.1-5.13.1.1.3항에 따른다. 융점이 925°C 미만인 재료로 된 비상차단밸브는 내화시험이 요구된다.

**G3.6.1.2 생산시험(Unit Production Testing):** 설계 압력의 1.5배에서 수압시험. 설계 압력의 1.1배에서 시트/스템 누설시험. -55°C 미만 밸브의 최소 10%에 대한 극저온시험.

### G3.6.2 벨로즈
IGC Code 5.13.1.2.1-5.13.1.2.4항에 따른 원형시험.

### G3.6.3 화물 펌프 및 가스/재액화/냉동 압축기

장비는 UR E10 및 UR M46을 고려하여 해양 환경에 적합해야 한다.

**G3.6.3.1 화물 펌프:**
(a) -55°C 미만의 경계 구성품에 대해 IGC Code 6.2.2항에 따른 재료시험.
(b) 원형시험: 설계 압력의 1.5배 수압시험, 성능시험.
(c) 생산시험: 설계 압력의 1.5배 수압시험, 성능시험.

**G3.6.3.2 가스 화물 및 재액화/냉동 압축기:**
(a) -55°C 미만의 경계 구성품에 대해 IGC Code 6.2.2항에 따른 재료시험.
(b) 원형시험: 수압시험, 기계적 운전시험, 성능시험.
(c) 생산시험: 설계 압력의 1.5배 수압시험.
(d) 설치: 완전 조립체의 누설시험.

## G3.7 배관 제작 및 접합 세부사항

### G3.7.2 직접 접합
IGC Code 5.8.2.1-5.8.2.3항에 따른다.

### G3.7.3 플랜지 접합
용접 넥형, 슬립온형 또는 소켓 용접형.

### G3.7.6 용접, 용접후 열처리 및 비파괴시험(NDT)
(a) IGC Code 5.9항 및 선급 요건에 따른다.
(b) 탄소강/탄소망간강/저합금강의 모든 맞대기 용접부에 대한 용접후 열처리(PWHT). 벽두께 10mm 미만의 경우 면제될 수 있다.
(c) -10°C 미만에서 내경 75mm 초과 또는 벽두께 10mm 초과인 배관에 대해 100% 방사선/초음파 검사. 기타의 경우 최소 10%.

## G3.8 선내 시험

### G3.8.2 압력시험
IGC Code 5.13.2.2항(강도), 5.13.2.3항(누설), 5.13.2.4항(이중벽 가스연료)에 따른다.

### G3.8.3 기능시험
최초 적재 시 정상 운전 조건에서 모든 배관 시스템을 시험한다.

IGC Code 5.12.3.1항 및 5.12.3.2항에 따른다.
