# IACS UR A 시리즈 한글 요약

---

## UR A2 — 일반 선박의 예인 및 계류 관련 선상 부속품 및 지지 선체 구조

**적용 대상:** 500 GT 이상의 일반 배수량형 선박 (고속선, 특수목적선, 해양구조물 제외)

### A2.0 적용 범위 및 정의

- 선박의 통상적인 예인 및 계류 작업에 사용되는 선상 부속품(bollard, bitt, fairlead, chock 등)과 지지 선체 구조의 설계 및 건조에 적용
- **통상 예인(Normal towing):** 항내 및 보호 수역에서의 조선 작업에 필요한 예인
- **기타 예인(Other towing):** SOLAS Reg. II-1/3-4에 따른 비상 시 예인 장비
- **적용 제외:** 호위 예인(Escort towing), 운하 통과 예인(Canal transit), 유조선 비상 예인(Emergency towing for tankers)
- 주요 정의:
  - **선상 부속품:** Bollard, bitt, fairlead, stand roller, chock 등 (윈치, 캡스턴 제외)
  - **지지 선체 구조:** 부속품이 설치되고 외력을 직접 받는 선체 구조부
  - **MBL_SD (선박 설계 최소 파단 하중):** 부속품 및 구조가 설계되어야 하는 새 건조 로프의 최소 파단 하중

### A2.1 예인 (Towing)

| 항목 | 내용 |
|------|------|
| **배치** | 부속품은 갑판 구조의 보강재/거더 위에 설치하여 하중을 효율적으로 분배 |
| **통상 예인 설계 하중** | 최대 예인 하중(정적 볼라드 풀)의 **1.25배** |
| **기타 예인 설계 하중** | IACS Rec. No.10에 따른 MBL_SD |
| **허용 응력 (빔 이론)** | 수직응력: 1.0 R_eH, 전단응력: 0.6 R_eH |
| **허용 응력 (FEA)** | Von Mises 응력: 1.0 R_eH |
| **안전 예인 하중 (TOW)** | 설계 하중의 80% 이하, 각 부속품에 용접 비드 등으로 표시 |

- 라인이 부속품에서 방향 전환 시, 부속품에 작용하는 설계 하중은 두 방향 라인 하중의 합력 (최대 라인 설계 하중의 2배 이하)
- Bollard/bitt의 예인 라인 부착점: 튜브 높이의 4/5 이상

### A2.2 계류 (Mooring)

| 항목 | 내용 |
|------|------|
| **부속품 설계 하중** | MBL_SD의 **1.15배** |
| **윈치 지지 구조 설계 하중** | 최대 브레이크 유지 하중의 **1.25배** (브레이크 유지 하중 ≥ MBL_SD의 80%) |
| **캡스턴 설계 하중** | 최대 인입력의 **1.25배** |
| **허용 응력** | 예인과 동일 (빔 이론: 1.0 R_eH / 0.6 R_eH, FEA: 1.0 R_eH) |
| **안전 작업 하중 (SWL)** | MBL_SD 이하, 각 부속품에 표시 |

- 계류 bitt는 8자형(figure-of-eight) 방식으로 라인을 감는 경우를 기준으로 선정
- 핀(fin)이 있는 bollard는 핀 위치를 라인 부착점으로 적용 가능

### A2.3 예인·계류 배치도

- 각 부속품의 SWL/TOW, 위치, 유형, 용도, 사용 방법(제한 편향각 포함)을 명시
- 계류 라인 배치, MBL_SD, 허용 환경 조건(풍속, 조류 속도) 포함
- 도선사 카드(Pilot card)에도 해당 정보 포함

### A2.4 부식 여유 (Corrosion Addition, t_c)

- CSR 적용 선박: CSR 규정에 따름
- 기타 선박: 주변 구조에 따른 선급 규정 적용, 갑판 위 받침대/기초 및 비표준 부속품은 **2.0 mm**

### A2.5 마모 여유 (Wear Allowance, t_w)

- 비표준 부속품의 라인 접촉면: 최소 **1.0 mm** 추가

### A2.6 건조 후 검사

- 부속품, 받침대/기초, 인접 선체 구조의 상태를 선급 규정에 따라 검사
- **Rev.5 적용:** 2022년 1월 1일 이후 건조 계약 선박

## UR A3 — 양묘기(Anchor Windlass) 설계 및 시험

### 1. 일반 사항

- **적용 대상:** 앵커 취급용 양묘기(windlass)
- **준거 기준:** SNAME T&R Bulletin 3-15, ISO 7825, ISO 4568, JIS F6714 등 인정된 표준에 적합해야 함
- **적용 시기:** 2018년 7월 1일 이후 인증 신청 또는 건조 계약 선박 (Rev.1은 2020년 7월 1일부터)
- 양묘기 기초 구조(하부 갑판 지지 구조)는 본 UR에 포함되지 않음 (UR S27의 4.2 참조)

### 2. 재료 및 제작

- 토크 전달 및 하중 지지 부품의 재료는 선급 규정 또는 국제 재료 표준에 적합
- 용접 절차 및 용접사는 선급 요건에 따라 자격 인증 필요
- 용접 소모품은 IACS UR W17/W23에 따라 승인

### 3. 설계

#### 3.1 기계 설계

| 항목 | 요구사항 |
|------|----------|
| **유지 하중(Holding Load)** | 체인 최소 파단 강도의 **80%** 하에서 항복 응력 이하 (체인 스토퍼 설치 시 **45%**) |
| **연속 사용 인력(Continuous Duty Pull)** | 체인 등급별 공식에 따라 최소 **30분** 지속 가능 |
| **과부하 능력** | 연속 인력의 **1.5배**를 최소 **2분** 이상 발휘 |
| **인양 속도** | 평균 체인 속도 ≥ **0.15 m/s** |
| **브레이크 용량** | 체인 최소 파단 강도의 **80%** (스토퍼 있으면 45%)에서 영구 변형 없이 유지 |
| **체인 스토퍼** | 체인 최소 파단 강도의 **80%**에 영구 변형 없이 견딤 |

#### 연속 인력 계산식

| 체인 등급 | Z_cont1 (N) | Z_cont1 (kgf) |
|-----------|-------------|----------------|
| 1등급 | 37.5d² | 3.82d² |
| 2등급 | 42.5d² | 4.33d² |
| 3등급 | 47.5d² | 4.84d² |

- 정박 수심 82.5m 초과 시: Z_cont2 = Z_cont1 + (D − 82.5) × 0.27d²

#### 3.2~3.5 기타 설계 요건

- **유압 시스템:** 선급 규정 준수
- **전기 시스템:** 100 kW 이상 모터 및 기어는 선급 인증 필요, 해상 노출부는 적절한 보호 등급
- **기계 부품 보호:** 과속 및 과토크 방지 장치 설치, 유압 모터 파손 시 파편 억제 고려
- **커플링:** 케이블 리프터와 구동축 분리 가능, 유압/전기식은 수동 분리도 가능해야 함

### 4. 공장 검사 및 시험

| 시험 | 내용 |
|------|------|
| **무부하 시험** | 정격 속도로 양방향 총 30분 운전 (기어 변속 시 추가 5분/방향) |
| **부하 시험** | 연속 인력, 과부하 용량, 인양 속도 확인 |
| **브레이크 시험** | 유지력을 시험 또는 계산으로 검증 |

### 5. 선상 시험

- 설치 후 실제 작업 조건에서 시험 수행
- 브레이크, 클러치, 체인/앵커 하강·인양, 체인 리프터 및 호스 파이프 통과, 적재 확인
- 평균 인양 속도(≥ 0.15 m/s) 측정 및 확인

### 6. 표시

- **영구 표시 사항:**
  - 양묘기 공칭 크기 (예: 100/3/45 = 100mm 직경, 3등급 체인, 유지 하중 45%)
  - 최대 정박 수심 (m)
