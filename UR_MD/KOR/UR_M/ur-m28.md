# M28 Ambient reference conditions
(1978)

For the purpose of determining the power of main and auxiliary reciprocating internal combustion engines, the following ambient reference conditions apply for ships of unrestricted service:

| Parameter | Value |
| :--- | :--- |
| Total barometric pressure | 1000 mbar |
| Air temperature | $$+45^{\circ}C$$ |
| Relative humidity | 60% |
| Sea water temperature (charge air coolant-inlet) | $$32^{\circ}C$$ |

**NOTE**
The engine manufacturer shall not be expected to provide simulated ambient reference conditions at a test bed.
